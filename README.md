# Kommit

> Git commits for the disillusioned human being

## Installation

### MacOS

- Homebrew

```bash
brew install madflow/kommit
```

### Docker

```bash
docker run -it --rm -v $PWD:/workdir madflow/kommit
```

## Usage

### Yolo mode

- No interaction
- Adds all changes
- Commits them
- Pushes them

```bash
kommit --yolo
```
